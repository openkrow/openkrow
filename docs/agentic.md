# OpenKrow must be the AI agents
OpenKrow is an agents AI agent that can help you complete all kinds of tasks. It is designed with a multi-agent architecture, where each agent is responsible for a specific task. This allows for better modularity and scalability, as new agents can be added or existing ones can be modified without affecting the overall system.
