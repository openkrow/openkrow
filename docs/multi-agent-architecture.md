# OpenKrow Multi-agent Architecture

## Knowledge
Use notebookllm to store knowledge and information. It can be used to store notes, documents, and other types of information that the assistant can access when needed.

## Multi-agent architecture
1. Engineer agent: responsible for technical tasks, such as coding, debugging, and system maintenance.
2. Researcher agent: responsible for gathering information, conducting research, and providing insights on various
3. Officer agent: responsible for office tasks, such as constructing documents, google sheet, email, calender, and other administrative tasks.
4. System agent: responsible for manage the overall system, setting openkrow, command and other system-level tasks.
