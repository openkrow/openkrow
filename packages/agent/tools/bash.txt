Execute a bash command in a shell session with optional timeout.

OS: ${os}, Shell: ${shell}

All commands run in the working directory by default. Use the `workdir` parameter to run in a different directory.

IMPORTANT:
- Default timeout is 2 minutes (120000ms). Use the `timeout` parameter for longer commands.
- Always quote file paths that contain spaces with double quotes.
- When issuing multiple independent commands, describe them clearly.
- If commands depend on each other, chain them with '&&'.
- ${chaining}
- Provide a clear, concise description of what the command does in 5-10 words.
